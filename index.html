Hexo GitHub 搭建静态技术博客一

发表于 2017-07-03 | 分类于 工具 ， Hexo

Hexo搭建技术博客大体可以分为以下几步

一、环境安装

github

1、在github注册账号，并且新建一个仓库，用于存放静态网站，仓库名称请用yourname.github.io,其中(yourname为您的github账户用户名)
2、安装Git客户端，您可以去Git下载，也可以点击博主个人云盘下载。密码：l2ox
3、安装TortoiseGit(可选)，此处可根据个人喜好，选择安装或者不安装，主要提供git的可视化操作，提高效率

nodejs

1、安装NodeJs客户端，你可以去NodeJs下载，也可以点击博主个人云盘下载。密码：dyi7

二、域名配置(可选)

1、域名购买–可以选择在阿里云上面购买域名
2、配置域名解析，配置域名解析的方法有好多种，一般选用CNAME解析
3、在GitHub博客首页所在根目录，增加CNAME文件，在里面添加你自己的域名即可(目的是使网页能够解析为自己制定的域名)；例如：在CNAME文件中添加www.yourname.cn

三、Hexo主题/配置

1、Hexo主题：登录Hexo主题官网下载更多主题，博主选用Next主题
2、Hexo配置：Hexo主要有两个主要的配置文件，一个是hexo工作目录下面的_config.xml,称为站点配置文件；一个是theme下面的_config.xml文件，称为主题配置文件；
站点配置文件主要配置网站公用的，主题配置文件主要配置自己的个性化显示。Next主题中基本上所有的配置可以在这两个配置文件中完成。
3、Hexo插件：hexo具备丰富的插件集成功能，可以通过第三方插件，丰富博客的功能。

三、Hexo插件

hexo快捷部署；

1、该插件使用 hexo d 直接可以部署到github上面(需在_congig.xml站点配置文件中配置deploy属性，配置如下，注意空格)

2、站点配置_config.xml配置如下
deploy:
type: git
repo:
github: git@github.com:yourname/yourname.github.io.git,master

站长地图

1、站长地图是能否使百度或者谷歌能够site到我们网站的一重要工具。

2、站点配置_config.xml配置如下
baidusitemap:
path: baidusitemap.xml
sitemap:
path: sitemap.xml

3、网易跟帖插件（需修改–详细点）
登陆 网易云跟帖 获取你的 Product Key。 编辑 主题配置文件， 编辑 gentie_productKey 字段，设置如下：

gentie_productKey: #your-gentie-product-key
请注意，您在云跟帖管理后台设置的域名必须跟您站点的域名一致。在本地测试时，需要做两步骤前置设定：
修改 hosts 文件，将您域名的请求指向本地。例如：127.0.0.1 notes.example.com
修改 Hexo 监听的端口为 80：hexo s –debug -p 80
测试完成后请将 hosts 文件中的域名映射删除
